# Simple Quadruped Robot Dog
This repository contains the mechanical design and report for a simple quadruped robot dog created during the
**Smart Methods Summer Training (July 2026)**.
---
## Project Overview
The project focuses on designing a simple robot dog that can stand, balance, and perform basic walking movements.

## Design Summary
- **Structure:** 4 Legs (Symmetrical Design)
- **Degrees of Freedom (DOF):** 8 DOF (2 joints per leg: Hip & Knee)
- **Actuators:** MG996R Servo Motors
- **Estimated Torque:** 0.5 N·m
- **Walking Method:** Trot Gait
